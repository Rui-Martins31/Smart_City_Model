** SETUP **

-> Para que a ESP se conecte a uma rede WiFI, o ficheiro exemplo "config.h_template" deve ser alterado de acordo com os dados de cada um. Isto inclui: o nome da nossa rede Wifi e a sua palavra-passe. (!!! Não alterar diretamente no código "connectToDatabase" !!!)

-> Guardar o novo ficheiro alterado como "config.h". Devemo-nos certificar que foi guardado em formato ".h". O resto já está configurado e, portanto, os nossos dados não serão expostos na nuvem/git. (!!! Não guardar as alterações no ficheiro "config.h_template" !!!)
